# Уроки Криейтора из правок владельца

> Живое состояние: устойчивые правила письма, извлечённые из того, как владелец
> доредактирует посты перед публикацией. Грузится в контекст Криейтора к каждому посту.
> Новое перекрывает старое; неудачный урок владелец удаляет руками. Личность (SKILL.md)
> правит только Девелопер через гейт — устоявшийся урок поднимают туда отдельно.

- (2026-06-17) Паттерн «Это не X. Это Y» (две параллельные фразы с «Это») — ВЫРЕЗАТЬ всегда. Скажи Y прямо, с фактом. Это ИИ-пул-квота, владелец требует убрать насовсем (линтер ловит). — _из правки: убрал «Это не сценарий из 2030-го. Это уже посчитано.» и «Это не энтузиасты в Discord.»_
- (2026-06-17) Throat-clearing-связки «И честно:», «честно говоря», «скажу прямо:» перед фактом — убирать. Факт честен сам по себе, подводка-уверение лишняя. — _из правки: «И честно: биткоин не дорожает» → «Биткоин с этого напрямую не дорожает»_
- (2026-06-17) Заголовок — точная теза-вопрос, без кликбейт-драмы и императива-подводки. — _из правки: «Крипта проиграла AI? Посмотрите, кто кому платит» → «Крипта - финансовая инфраструктура для AI?»_
- (2026-06-17) Закрывающий вопрос — прямой и в настоящем времени, без гипотетической подводки «Если через 5 лет… Вы хотите…». — _из правки: «Если через пять лет половина платежей… Вы хотите быть в активе…» → «Для Вас крипта как инфраструктура для экономики машин - уже реальность или красивая теория?»_

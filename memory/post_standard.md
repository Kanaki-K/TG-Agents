## Каталог форматов (тех-карты)

> Каждый формат — **тех.карта**, как рецепт в общепите. Криейтор: **видит карту → берёт данные
> (бриф Скаута, метрики Аналитика) → пишет по карте → проверяет себя по карте → отдаёт драфт на
> утверждение владельцу.** Формат под тему выбирается по данным (плейбук Аналитика), а не по умолчанию.
>
> Ориентиры (средние/пост по каналу): флагман 🔁9 · короткий ER 2.2 · психология ER 2.4 ·
> личный ER 2.6 + комментарии · обучающий ровный · медиа — просмотры есть, вовлечённости нет.

### Ф1 — Флагман (глубокий разбор + медиа) 🏆
> Полная спека флагмана — в [content_manual.md](content_manual.md) §2–§5 (мастер-документ). Здесь — кратко.
- **Что это:** медиа + глубокий авторский разбор, завершённая экспертная мысль. Главный пост, ~раз в неделю (четверг 16–19:00).
- **Для чего:** охват и вирусность — **репостов ~9/пост (в 4–5 раз больше любого другого формата)**,
  просмотры ~730, комментарии ~5. Расходится по чужим каналам.
- **Размер:** **2200–2500 знаков** (рабочий; диапазон 1500–3000). НЕ «чем длиннее, тем глубже» — драфт
  на ~580 слов был вдвое больше нормы. Длина — следствие глубины, не самоцель.
- **Структура:** **чередование «думающий → знающий»** (Д-Зн-Д-Зн-…): хук-вопрос → цифра → мысль →
  макро-данные → перевод → BTC-метрики → главная мысль → личное действие → вопрос аудитории. Подробно — мануал §4.
- **✅:** «Вы» с большой буквы; конкретные цифры с источниками + привязка к моменту; личная фраза;
  1–2 эмоц. приёма; эмодзи-разделы; **медиа**; футер.
- **❌:** вода/повтор хука; ссылки в тексте; AI-паттерны (§7 мануала); $ перед числом; длинные тире, ёлочки.
- **Примеры:** #434 «Смена рук на дне рынка», #417 «Сокращение портфеля 10→5», #432 «17 лет крипте».
- **Метрика-ориентир:** репосты (вирусность). ER ниже из-за высоких просмотров — это нормально.

### Ф2 — Обучающий
- **Что это:** объяснение концепции/механики простым языком (шире «словаря» — любой разбор «как
  устроено»). ~300 слов.
- **Для чего:** польза и охват (просмотры ~660); ровный рабочий формат, держит экспертность.
- **Когда:** Пятница (образовательный слот) или под обучающую тему.
- **Структура:** понятие/вопрос → простое объяснение → пример/аналогия → зачем это держателю.
- **✅:** объяснять жаргон; аналогии из жизни; один разбор — одна мысль.
- **❌:** узкий жаргон без объяснения; сухая энциклопедия без «зачем».
- **Примеры:** #416 «Как хайп убил здравый смысл в 2017», #378 «Приватность в крипте: основы».
- **Метрика-ориентир:** просмотры + ER ~1.8%.

### Ф3 — Психология / мышление
- **Что это:** короткий пост про поведение инвестора, дисциплину, ловушки мышления. ~150 слов.
- **Для чего:** **высокая вовлечённость (ER ~2.4%)**; держит ритм, бьёт в боль аудитории.
- **Когда:** начало недели / ритмовые слоты.
- **Структура:** тезис о поведении → разбор ловушки/механики мышления → установка/вывод.
- **✅:** философский слой («зачем это в жизни»); самоирония; короткие предложения, воздух.
- **❌:** пустая мотивация без мысли; обещания/призывы.
- **Примеры:** #435 «DCA как формула жизни», #421 «Не переделывайте портфель в понедельник», #262.
- **Метрика-ориентир:** ER.

### Ф4 — Личный / дневниковый
- **Что это:** от первого лица: своё решение, путь, честная отчётность (вкл. ошибки). ~150 слов.
- **Для чего:** **топ по вовлечённости и обсуждаемости (ER ~2.6%, больше всего комментариев)**;
  доверие ядра — «личный дневник» как часть жанра бренда.
- **Когда:** когда есть реальный личный кейс (докупил/продал/пересмотрел/ошибся).
- **Структура:** личная ситуация/решение → что сделал и почему (честно) → вывод/урок читателю.
- **✅:** признание ошибок; «я держу / я докупил…»; конкретика; вопрос аудитории (зовёт коммент).
- **❌:** хвастовство; **точные $-суммы дохода/портфеля** (красная линия — активы и % можно, $ нет).
- **Примеры:** #433 «Откуп при падении на 13%», #430 «5-я DCA покупка», #364 «XMR продал 30%».
- **Метрика-ориентир:** ER + комментарии (обсуждение).

### Ф5 — Короткий (свежее со своим углом)
- **Что это:** короткий пост с авторским углом на свежее событие/цифру. ~250 слов.
- **Для чего:** оперативность и темп между флагманами; рабочий ER ~2.2%.
- **Когда:** есть свежая цифра/новость, но не на флагман.
- **Структура:** факт/цифра → твой угол (что это значит) → короткий вывод.
- **✅:** ОБЯЗАТЕЛЬНО свой угол/разбор; конкретная цифра-крючок.
- **❌:** **чистая новость без разбора** (мёртвый формат); ссылки в тексте.
- **Примеры:** #436 «IPO SpaceX и отток из крипты», #431 «Месячная пауза и нарратив».
- **Метрика-ориентир:** ER + просмотры.

### Бакеты (не контент-карты, но нужны для учёта)
- **Медиа** — пост из медиа почти без текста. Просмотры высокие (~780, «свежесть»), но **ER мёртвый
  (~1.07%)**. Правило: **не публиковать голое медиа** — добавляй содержательный текст (тогда это Ф1/Ф5).
- **Служебное** — розыгрыши, уровни канала, итоги-анонсы. Не контент-формат, в плейбуке не оптимизируем.

> Список форматов — живой: новые форматы-кандидаты предлагает Аналитик в плейбуке по данным/трендам,
> Криейтор оформляет их как тех-карту через `/derive` (на утверждение владельцу).

## Что делает пост сильным (общее для всех форматов)
- **Авторское мнение, не копипаста.** Не «квартал упал», а «почему упал, как оценить, что значит».
- **Структура факт → контекст → вывод.** Начать с цифры/факта → логика/история → вопрос или вывод.
- **Личный опыт как основа.** «я пересмотрел портфель и вот почему», «я держу BTC/ETH/…».
- **Конкретика.** Не «рынок падает», а «BTC −23%, ETH −32%, золото +19%». Цифры, сравнения, аналогии.
- **Философский слой.** Зачем это в жизни, а не только в портфеле.
- **Контрарность + личная история.** Переворот общепринятого через свой опыт (заходит лучше всего).
- **Самоирония; короткие предложения и воздух** между блоками.

## Анти (❌ — не работает по данным; общее)
- Чистые новости без авторского разбора — слабый ER.
- **Голое медиа без текста — ER ~1.07% (мёртвый).** Картинка без пояснения не работает.
- Узкий жаргон без объяснения; прямые сигналы «купи/продай».
- Вода ради объёма; ссылки в теле поста — убивают охват (источники только в брифе, нам, на проверку).

## Ритм недели (узнаваемый аудитории)
- **Пн, 5:00** — Ф3 психология / лёгкое, ритм-старт.
- **Середина недели (13–14:00)** — основной контент: Ф1 флагман / Ф4 личный / Ф5 короткий.
- **Пятница, 14:00** — Ф2 обучающий.
- Время: пик 14:00, но **не частить** в один час — ER по перегруженным слотам падает.

## Текст vs медиа
По данным: **медиа даёт просмотры** (свежесть, ~780), **текст с выводами даёт вовлечённость** (ER).
Голое медиа — мёртвый ER. Лучшее — **медиа + содержательный текст** (это и есть флагман), не медиа вместо текста.

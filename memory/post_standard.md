# Стандарт поста — KANAKI CRYPTO (Telegram)

> Как писать пост на канале **Telegram**. Спецификация для **Криейтора**; **Скаут** сверяется с ней,
> чтобы искать темы, под которые получится сильный пост. Собрано из характеристики аналитика по
> 362 постам + уточнений владельца. Канон бренда, **тон и красные линии** — в [brand.md](brand.md)
> (соблюдать строго: BTC не проигравший, без политоты кроме прямого влияния на крипту и т.д.).
>
> **Threads — отдельный стандарт на Этапе 2** (другой формат: один флагман дистиллируется в ~4
> мини-флагмана). Стандарты ведём по платформам в разных файлах.
>
> Поля карт, помеченные `[уточнить]`, выведены по смыслу, а не из данных — их доуточняет владелец
> или Криейтор командой `/derive` (объёмы, примеры id). «Что заходит» по цифрам ведёт Аналитик в
> [format_playbook.md](format_playbook.md); здесь — **как писать** каждый формат.

## Каталог форматов (тех-карты)

> Каждый формат — **тех.карта**, как рецепт в общепите. Криейтор: **видит карту → берёт данные
> (бриф Скаута, метрики Аналитика) → пишет по карте → проверяет себя по карте → отдаёт драфт на
> утверждение владельцу.** Формат под тему выбирается по данным (плейбук Аналитика), а не по умолчанию.

### Ф1 — Флагман (глубокий разбор) 🏆
- **Что это:** длинный авторский разбор темы с медиа, 800–1100 слов.
- **Для чего:** главный «удар» канала — то, что реально шарят (20+ репостов, расходится по чужим каналам).
- **Когда:** середина недели (13–14:00); сильная глубокая тема (макро/механика/исторические кейсы).
- **Объём:** 800–1100 слов. ⚠️ Потолок ~1100: #425 на 1200+ собрал меньше репостов — «глубже» ≠ «длиннее».
- **Структура (шаблон #434):**
  1. **Заголовок-вопрос** — то, что цепляет (~20 слов).
  2. **Вводный факт** — что происходит (~50 слов).
  3. **Механика** — как/почему это работает (~200 слов).
  4. **Кейсы** — исторические примеры (~500 слов).
  5. **Личный вывод** — что это значит для держателя, что дальше (~100 слов).
  6. Картинка/график. ≈ 900 слов + медиа.
- **✅:** факт → контекст → вывод; конкретные цифры; исторические кейсы; философский слой.
- **❌:** вода ради длины (>1100 слов); ссылки в тексте; пересказ без своего угла.
- **Примеры:** #434 (31 репост), #417 (29).
- **Ориентир-метрика:** репосты 20+.

### Ф2 — Личный / дневниковый
- **Что это:** честный пост от первого лица про свой опыт, решения и **ошибки**, 300–600 слов `[уточнить]`.
- **Для чего:** доверие и лояльность ядра; «личный дневник» — часть жанра бренда; контрарность через
  свою историю заходит лучше всего (по реф-анализу).
- **Когда:** когда есть реальный личный кейс/решение (пересмотрел портфель, ошибся, поменял взгляд).
- **Структура:** ситуация/триггер → что сделал и почему → честный итог (включая провал) → урок для читателя.
- **✅:** признание ошибок; «я держу / я пересмотрел…»; конкретика; без позы гуру.
- **❌:** хвастовство; **точные $-суммы дохода/портфеля** (красная линия — активы и % можно, доллары нет);
  «мотивашка» без личного факта.
- **Примеры:** `[уточнить id]`.
- **Ориентир-метрика:** ER (вовлечённость), комментарии.

### Ф3 — Лёгкий философский / мотивационный
- **Что это:** короткий рефлексивный текст, держит ритм, 200–400 слов, текст.
- **Для чего:** «энергетический» формат, лёгкая вовлечённость — ритм недели, не главный удар.
- **Когда:** Пн, 5:00.
- **Структура:** мысль/образ → разворот → вопрос или вывод.
- **✅:** философский слой («зачем это в жизни»); самоирония; короткие предложения, воздух.
- **❌:** пустая мотивация без смысла; обещания/призывы.
- **Примеры:** #256, #262.
- **Ориентир-метрика:** ER, охват за счёт ритма.

### Ф4 — Короткий пост (свежее со своим углом)
- **Что это:** короткий пост с авторским углом на свежее событие/цифру, 150–350 слов `[уточнить]`.
- **Для чего:** оперативность, темп между флагманами.
- **Когда:** есть свежая цифра/новость, но не на полноценный флагман.
- **Структура:** факт/цифра → твой угол (что это значит) → короткий вывод.
- **✅:** ОБЯЗАТЕЛЬНО свой угол/разбор; конкретная цифра-крючок.
- **❌:** **чистая новость без разбора** (ER ~0.5%, мёртвый формат); ссылки в тексте.
- **Ориентир-метрика:** просмотры/охват.

### Ф5 — Образовательный «Словарь крипто»
- **Что это:** разбор одного понятия/механики простым языком.
- **Для чего:** **топ по ER (~6–7%, кратно выше среднего)** — сильнейший образовательный формат.
- **Когда:** Пятница, 14:00 (образовательный слот).
- **Структура:** термин → простое объяснение → пример/аналогия → зачем это держателю.
- **✅:** объяснять жаргон; аналогии из жизни.
- **❌:** узкий жаргон без объяснения; сухая энциклопедия без «зачем».
- **Ориентир-метрика:** ER ~6–7%.

> Список форматов — живой: новые форматы-кандидаты предлагает Аналитик в плейбуке по данным/трендам,
> Криейтор оформляет их как тех-карту через `/derive` (на утверждение владельцу).

## Что делает пост сильным (общее для всех форматов)
- **Авторское мнение, не копипаста.** Не «квартал упал», а «почему упал, как оценить, что значит».
- **Структура факт → контекст → вывод.** Начать с цифры/факта → логика/история → вопрос или вывод.
- **Личный опыт как основа.** «я пересмотрел портфель и вот почему», «я держу BTC/ETH/…».
- **Конкретика.** Не «рынок падает», а «BTC −23%, ETH −32%, золото +19%». Цифры, сравнения, аналогии.
- **Философский слой.** Зачем это в жизни, а не только в портфеле.
- **Контрарность + личная история.** Переворот общепринятого через свой опыт (заходит лучше всего).
- **Самоирония; короткие предложения и воздух** между блоками.

## Анти (❌ — не работает по данным; общее)
- Чистые новости без авторского разбора — ER ~0.5%.
- Пустые медиа / картинка без пояснения — ER ~0.06–0.16%.
- Узкий жаргон без объяснения; прямые сигналы «купи/продай».
- Перебор объёма (1200+ слов) — репостов меньше.
- **Ссылки в теле поста — убивают охват.** Источники нужны только в брифе (нам, на проверку), НЕ в тексте.

## Ритм недели (узнаваемый аудитории)
- **Пн, 5:00** — Ф3 лёгкий философский/мотивационный.
- **Середина недели (13–14:00)** — основной контент: Ф1 флагман / Ф2 личный / Ф4 короткий.
- **Пятница, 14:00** — Ф5 образование («Словарь крипто»).
- Время: пик 14:00, но **не частить** в один час — ER по перегруженным слотам падает (~1.34% vs 1.96%).

## Текст vs медиа
Текст с выводами даёт вовлечённость (ER), медиа — просмотры за счёт «свежести». Лучшее —
**медиа + содержательный текст**, не медиа вместо текста.

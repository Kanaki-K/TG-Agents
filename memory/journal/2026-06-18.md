## РЕШЕНО: контент-модель канала (зафиксировано владельцем, конец сессии с Аналитиком)

Долгая итерация с Аналитиком ушла в круг (он пересобирал таблицы заново и плавал по цифрам).
Решения владельца ниже — финальные, перенесены в `content_manual.md` (§2, §9) и `post_standard.md`
(«Ритм недели»). Криейтор берёт оттуда, переобсуждать не нужно.

### Частота и дни
- **2 флагмана в неделю: вторник + четверг.** Причина — РИТМ владельца («лучше» = когда он постил, а
  не день по статистике). Решено, не оптимизировать по дням.
- **Короткий со своим углом — Пн / Ср / Пт** (тестируем).
- **Выходные (Сб/Вс) — постов нет.**
- Итого: 4–5 плановых постов/нед + личное вне графика.

### Флагман = ФОРМАТ, не тема
- Длинный пост (одно сообщение TG) = флагман. Темы варьируются: макро/стратегия, психология+стратегия,
  глубокий разбор технологии/ниши (DeFi/RWA/AI/DePIN/L0–L3), личный опыт. Вт и Чт — разные углы.

### Новый/уточнённый тип: короткий со своим углом (Ф5)
- Важное рыночное событие как ЛИЧНОЕ наблюдение/мнение, не трансляция новости. 150–250 слов.
  Факт → почему важно для инвестора → сигнал/вывод. Чистые новости = отдельный канал.

### Вне графика
- DCA-дневник и личные наблюдения — по факту события (покупка/продажа/ребаланс), не по расписанию.

### ДРОП
- Выходные; формат «Доброе утро»; чистая новость-трансляция; голое медиа без текста.

### Приоритет метрик (поправка владельца, для Аналитика)
1. Просмотры (смотрят ли вообще) → 2. Лайки (вовлечение) → 3. Репосты (ценность/вирусность) →
4. Комменты (редко, но честно). ER вторичен; смотреть абсолютные лайки, не только %.

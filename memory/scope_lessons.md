# Уроки 🔭 «Под прицелом» из правок владельца

> Живое состояние: устойчивые правила КОРОТКОГО аналитического поста, выловленные из того,
> как владелец доредактирует 🔭-посты перед публикацией. Грузится в контекст scope-ветки
> (core/scope_writer) к каждому короткому посту — ОТДЕЛЬНО от флагман-уроков (post_lessons.md).
> Новое перекрывает старое; неудачный урок владелец удаляет руками. Правила рубрики (длина,
> подача, красные линии) живут в scope_manual.md — сюда идёт только то, что выловлено из ПРАВОК.

- (2026-06-30) ОСЬ scope = НОВОСТЬ + ТВОЙ ЕДЖ. Новость коротко и ВЛАДЕЙ ею (говори от себя, не «лента прочитала это как»). Едж ОБЯЗАТЕЛЕН и не банальный: деньги-механика с пейоффом / масштаб-рамка / value-accrual до холдера / оптика долгосрочника. Нет острого угла к холдеру — повод НЕ для 🔭. — _из разбора MSTR-поста 30.06: новость была, едж тонкий («научилась пересиживать»), подача журналистская_
- (2026-06-30) ПОДАЧА scope — живой уверенный разбор от первого лица, НЕ журналистика. ❌ «Лента прочитала это как…», «фирма прописала, что делать», «это важнее, чем кажется». ✅ владеешь фактом, разговорно («будут гонять», «для контекста»), приземлённый образ с иронией («вцепились за жирный кусок»), закрытие как выдохнул. Тон бери из K1/K2, НЕ из a01k (у него плоский новостной — только длину). — _из разбора MSTR-поста 30.06_

# Реестр источников для Скаута

> Кого Скаут сканирует и кому доверяет. Тиры — по тому, насколько источник **первичен**.
> Источники разные «руки»: RSS/веб дёшево, видео/подкаст и платные API — дорого.
> ⚠ URL-ы фидов помечены «проверить» — сверяем перед тем, как подключать в коннектор.

## Зачем тиры (каждый отвечает на свой вопрос)
- **Тир 1 — первоисточники:** *это вообще реально?* (ранний сигнал, но дорого/сложно)
- **Тир 2 — мысль/тезисы:** *какой умный угол?* (глубина; часто просто RSS/веб — дёшево)
- **Тир 3 — агрегаторы:** *это уже разгоняется?* (скорость; поздно, но видно момент)

**Проверка достоверности = кросс-тир:** прослеживается ли хайп из Тир-3 до реального
источника в Тир-1/2, или это круговой перепост сам себя. Для крипты — фильтр от скама.

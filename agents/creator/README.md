# Агент: Криейтор

Пишет **готовые посты** по направлению от владельца (часто — по выводу Скаута/дип-ресёрча).
Продукт — драфт в голосе автора на правку. Замыкает «ресёрч → пост».

## Запуск
```
python run_creator.py
```
Команды: `/post` — флагман по сильнейшему направлению свежего брифа Скаута (или скажи, по какому);
`/light` — лёгкий философский (200–400 слов). Можно и просто прислать тему/направление.

## Конфиг
- Модель: `claude-opus-4-8` (качество письма критично).
- `thinking: adaptive` — острее композиция и соблюдение красных линий (убери строку, чтобы выключить).
- Бот-токен: `CREATOR_BOT_TOKEN`. Ключ: `CREATOR_ANTHROPIC_KEY` (или общий).

## Инструменты (`core/creator_tools.py`)
Узкие, под письмо:
- `read_brief` / `list_briefs` — забирают бриф Скаута напрямую из `memory/briefs/` (стык «ресёрч → пост»).
- `find_posts` / `by_theme` — сверка с историей канала (анти-повтор, отсылки).
- `save_draft` — архив готового драфта в `memory/drafts/`.
В контекст также подаются `memory/brand.md` (голос/красные линии), `memory/post_standard.md`
(архетипы/флагман) и сводка канала. Публикации нет — драфт автомат, публикует владелец.

## Железные правила (из SKILL)
- Не выдумывает цифры: чего нет — ставит `[ПРОВЕРИТЬ: …]`.
- Ссылки НЕ в текст поста (убивают охват) — только в брифе.
- Красные линии бренда; один драфт за раз; публикует владелец.

## Формат
По умолчанию — флагман 800–1100 слов (заголовок-вопрос → факт → механика → кейсы → личный вывод).
По просьбе — лёгкий философский 200–400 слов.

## Файлы
- Данные: `agents/creator/{config.yaml, SKILL.md}`
- Код: `core/creator_bot.py`, `core/creator_tools.py`; канон — `memory/{brand.md, post_standard.md}`
- Вход: `memory/briefs/` (от Скаута) · Выход: `memory/drafts/`
